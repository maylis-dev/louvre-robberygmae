# Louvre Robbery Game

A browser-based heist game built with HTML, CSS, and JavaScript.  
Play as a robber trying to escape after selling stolen jewels from the Louvre.

---

## Game Description

The player takes the role of a robber selling jewels stolen from the Louvre.  
After being caught, the robber must escape without being arrested by the police.

### Rules:
- The player must collect 10 jewels to successfully escape
- If the player collides with the police, the game is over
- If the player collides with a tourist, the player loses points

The goal is to collect all jewels while avoiding the police and minimizing contact with tourists.
